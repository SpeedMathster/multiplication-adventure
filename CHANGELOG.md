## [2026-06-11] – Initial Repository & Core Game

### Added (Game Features – Full Release)
- **Quiz Mode** – Timed multiplication quiz with configurable question count, time limit, and specific table (or mixed). Score is weighted 80% accuracy / 20% speed.
- **Fill in the Blank** – Complete the missing number in an equation. Session tracks correct/wrong answers and remaining questions. Custom modal shows results with percentage and feedback.
- **Speed Drill** – 1 to 30 minute drill. Answer input and Submit button appear only after pressing Start Drill. Results shown in custom modal with weighted scoring.
- **Flashcards** – Digital flashcards that can show a specific table (1–max) or random mixed. Reveal answer on demand.
- **Skip Counting** – Choose a table (2 to current max) and see the full skip count sequence. Auto-updates when table changes.
- **Mastery Tab** – Displays every equation for a chosen table (e.g., 1×1=1 … 1×12=12) in a clean, centered list.
- **Pattern Explorer** – Five math learning patterns: 9’s finger trick, doubling trick, swap trick, 11’s magic, and tough ones mnemonics. Display text is large and readable.
- **Multiplication Chart** – Interactive grid (up to current max). Click on row/column numbers or individual cells to highlight related products. Reset highlights button.
- **Global Table Range** – Dropdown to set the maximum table (5 to 20). Affects all tools (Quiz, Flashcards, Skip Counting, Mastery, Chart).
- **Custom Result Modal** – Used for Quiz, Fill in the Blank, and Speed Drill. Shows accuracy %, speed %, total score, detailed stats, and three buttons: Play Again, Main Menu, Exit. Exit returns to the current tool without switching panels.
- **Visual Feedback Icons** – For correct/wrong answers in Quiz, Fill in the Blank, and Speed Drill (check mark or X with the correct answer hint).
- **Responsive Design** – Works on mobile devices with adjusted font sizes and layout.
